Intro to JavaScript — Variables & Data Types

Examples of var, let, const and common JS data types: number, string, boolean, null, undefined, object, array, symbol, bigint.

Variables: var, let, const

Click to run examples and see behavior.

// var (function-scoped), let (block), const (constant)
var a = 5;
let b = 10;
const c = 15;
// b = 20; // allowed
// c = 20; // error (can't reassign const)

Primitive Data Types

Numbers, Strings, Booleans, null, undefined, symbol, bigint

// Examples
let n = 42;              // number
let s = "Hello";         // string
let flag = true;         // boolean
let empty = null;        // null
let notSet;              // undefined
let sym = Symbol('id');  // symbol
let big = 12345678901234567890n; // bigint

Objects & Arrays

Compound data types (mutable).

// Object
const person = { name: "Omar", age: 21 };
// Array
let arr = [1, "two", true, {x:10}];

typeof & Type Conversion

Detect type and convert values.

// parseInt, Number, String, Boolean
parseInt("123") -> 123
Number("12.5") -> 12.5
String(123) -> "123"
Boolean(0) -> false
